# 国科财通 Steam 账号工具

Windows 平台下的 Steam 账号管理工具。

## 功能

### 辅助邮箱
- 导入原始账号和辅助邮箱
- 一对一匹配关联
- 导出合并数据

### 导出账号
- 导入 Steam 账号信息
- 导出为 txt 或 xlsx 格式

### 筛选令牌
- 选择账号文件和归属
- 从令牌库中筛选对应账号的令牌
- 将令牌复制到指定目录

## 使用方法

1. 运行 `国科财通steam账号工具.exe`
2. 根据需要选择相应功能标签页
3. 按照界面提示操作

## 环境要求

- Windows 系统

## 技术栈

- Python 3
- Tkinter (GUI)
- PyInstaller (打包)
